CausalCrash

A Hierarchical Benchmark for Evaluating Counterfactual Reasoning in Road Safety Scenarios

NeurIPS 2026 Evaluations & Datasets Track Submission #3454

Code 🤗 Dataset 📄 Paper (Coming Soon)

Abstract

Evaluating the causal and counterfactual reasoning capabilities of Vision-Language Models (VLMs) in complex, dynamic scenarios is a critical step toward reliable autonomous driving. We introduce CausalCrash, a hierarchical benchmark dataset consisting of high-severity real-world driving videos. Unlike existing datasets that focus on simple perception or what-happened QA, CausalCrash evaluates models across multiple cognitive layers: fine-grained perception and role assignment (Level 1), temporal hazard phase localization (Level 2), and deep causal/counterfactual reasoning (Level 3).

Full Example: Multi-Vehicle Complex Crash

This example features an overly speedy cement mixer truck entering an intersection, leading to a severe rollover and crushing a silver van. Let's compare the Ground Truth reasoning with the output generated by Gemini 2.5 Flash.

✅ Ground Truth (Human Expert) 
"causal_chain": [
  { "step": 1, "event": "Silver van travels steadily forward..." },
  { "step": 2, "event": "Mixer truck enters intersection from right at excessive speed." },
  { "step": 3, "event": "Mixer truck swerves sharply left to avoid direct impact." },
  { "step": 4, "event": "High-speed evasive maneuver causes top-heavy truck to tip over." }
],
"counterfactual_analysis": [
  {
    "actor": "mixer_truck",
    "action": "reduce_speed",
    "explanation": "If the mixer had approached at a safer speed, the driver could have braked straight, avoiding both the rollover and collision."
  }
]
🤖 Gemini 2.5 Flash 
"causal_chain": [
  { "step": 1, "event": "A heavy vehicle (cement mixer) attempts to turn on a bridge." },
  { "step": 2, "event": "A van approaches from the opposite direction." },
  { "step": 3, "event": "The truck appears to be swerving, losing control, and collides with the van." },
  { "step": 4, "event": "The truck overturns and the van is severely crushed." }
],
"counterfactual_analysis": [
  {
    "actor": "A",
    "action": "stay_in_lane",
    "explanation": "If the truck driver had stayed in their lane and not swerved, the collision with the oncoming van could have been avoided."
  }
]

Notice how Gemini 2.5 correctly identifies the actors and outcome, but falsely suggests "staying in lane" as a valid counterfactual, missing the critical nuance that the excessive speed made the rollover inevitable upon swerving.

Evaluation Pipeline

  1. 1

    Model Inference

    Vision-Language Models (e.g., GPT-4o, Gemini 2.5, InternVL) process the crash videos and output structured JSON annotations mimicking the three-tier schema.

  2. 2

    LLM-as-a-Judge Evaluation

    Advanced LLMs (GPT-4.1-mini and Gemini 2.5 Flash) compare the predicted JSONs against human ground-truth annotations across strict, lenient, and structural reasoning metrics.

  3. 3

    CausalCrash Score (CCS)

    Generation of an aggregated score that penalizes surface-level pattern matching and rewards deep temporal and counterfactual reasoning.
